Demo
0. Directly run visualize() or visualize_webcam() or plot(), fatal results will be generated.
1. Train model with small epochs or less images with train(), the model is bad.
2. run plot(), the training curve will be drawn.
3. Visualize the results with visualize() or visualize_webcam(), 
4. Train model with bigger epochs or more images with train(), the model will be better.
5. Continue running visualize() or visualize_webcam() or plot()
6. Run reset(), the whole system will be only trainable.


Customized
1. Directly run visualize() or visualize_webcam() or plot(), fatal results will be generated.
2. Trainable with train(), the data will be trained and the model will be generated in corresponding places.
3. run plot(), the training curve will be drawn.
4. Visualize the results with visualize() or visualize_webcam()
5. Train()/ plot() and visualize() or visualize_webcam() again, the algorithms will begin a new cycle
6. Run reset(), the whole system will be only trainable.
